# TabBar 图标说明

此目录需要放置 8 个 PNG 图标文件。

## 需要的文件

| 文件名 | 尺寸 | 颜色 | 用途 |
|--------|------|------|------|
| home.png | 81x81px | #999999 | 首页 - 未选中 |
| home-active.png | 81x81px | #4A90E2 | 首页 - 选中 |
| exercise.png | 81x81px | #999999 | 练习 - 未选中 |
| exercise-active.png | 81x81px | #4A90E2 | 练习 - 选中 |
| chat.png | 81x81px | #999999 | AI - 未选中 |
| chat-active.png | 81x81px | #4A90E2 | AI - 选中 |
| profile.png | 81x81px | #999999 | 我的 - 未选中 |
| profile-active.png | 81x81px | #4A90E2 | 我的 - 选中 |

## 图标获取方式

1. 从 UI 设计师获取
2. 使用 iconfont 图标库：https://www.iconfont.cn/
3. 使用 Figma/Sketch 绘制
4. 使用在线工具：https://icon.kitchen/

## 注意事项

- 图标需放在项目根目录的 `images/tabbar/` 下
- 构建后会自动复制到 `dist/images/tabbar/`
- 格式必须是 PNG
- 建议使用透明背景
